# 美正 Pack — 需求收集中心

> **使用方法**：你想到任何业务需求就丢进对应的小节，**一句话粗草稿就行，不用工整**。
> 等需求攒够（或者每周固定整理一次），我们一次性转成 Pack 的 manifest / workflow / skill / view。
>
> **粒度参考**：
> - 一句"客户问 91701 几号 zone 我要立刻知道" → 这是一个 **skill 查询**
> - "每月 1 号自动更新 UPS 全美分区表" → 这是一个 **定时 workflow**
> - "Home 首页要看到本周 UPS 燃油费率走势图" → 这是一个 **view（视图）**
> - "客户在群里 @ 我问运费报价" → 这是一个 **MCP 服务器（IM 集成）**
