## 30 秒电梯版（客户没耐心 / 群里发视频）

> 用于：朋友圈视频、给陌生卖家看一眼、群里炸鱼

| 秒 | 你说 / 演示 | 屏幕显示 |
|---|---|---|
| 0-3 | （打开软件，停在 Home 卡片上） | 跨境电商总览，3 个数字 + 3 按钮 |
| 3-8 | 对 AI 说：**"帮我看看昨天广告"** | AI 自动识别 → 跑 ad_daily_check |
| 8-15 | （AI 跑完，回 3 段格式） | 我看到的 / 我建议 / 你的决定 |
| 15-20 | 对 AI 说：**"换成 GLM 再说一次"** | LLM 切换胶囊变绿 + 重新跑 |
| 20-25 | 对 AI 说：**"如果我做财务呢"** | Pack 切换 → 整个 Home 卡片换 |
| 25-30 | 关 wifi。AI 继续工作。 | 顶部出现绿色"离线 · 数据 0 外发" |

**这 30 秒打了 4 张牌**：
1. 不绑模型（中立）
2. 不绑行业（Pack 可换皮）
3. 不绑云（本地 + 离线）
4. AI 不替你做决定（决策归还）

## 3 分钟主菜版（约到视频会议时）

### 第 0:00-0:20 · 一句话定位

> "我做了个本地装的 AI 控制台。**不绑模型、不绑行业、不上云**。
> 你只用动嘴，它替你操作所有东西。
> 数据全在你电脑里，可以拔网线给你看。
> 今天我用我朋友圈一个跨境电商朋友的真实场景演示。"

**关键词**：本地 / 不绑 / 动嘴 / 拔网线。

### 第 0:20-0:50 · 演示 1 · 对话操控（亮点 5）

打开 Chat。**不点任何按钮**。

> 我："帮我看看昨天广告数据"

AI 应回（3 段）：

```
🧠 我看到的：7 个 campaign · 总花费 $423 · ACOS 27.8% · 比目标高 2.8%

💡 我建议：
   • 暂停 SP-New-Launch-A14（花了 $18，0 单）
   • 降低 SP-ASIN-Competitor 的 bid（ACOS 55%）
   • 给 SP-Brand-Defense 加预算（ACOS 8.35%，CTR 3.23%）

👇 你的决定：
   [✓ 全部执行]  [改改再执行]  [我自己来]

   👀 [去 Tasks 页看完整报告 →]
```

**对客户说**：

> "你看，AI 给建议，按钮在你这。
> 它不会替你做决定，**方向盘永远在你手里**。"

### 第 0:50-1:20 · 演示 2 · 换模型（亮点 1 · 中立）

> 我："换成 GLM 再分析一次"

AI 应回：

```
✓ 已切换到 GLM-4.5-Flash
✓ 重新分析中…

[ DeepSeek · GLM ✓ · MiniMax · Anthropic ]   ← 顶部胶囊变绿

👀 [去 Models 页看完整模型库 →]
```

跑完之后看分析有什么差异（哪怕没差异也行）。

**对客户说**：

> "你今天用 DeepSeek 便宜，明天有了私有模型可以用私有的，后天 GPT-5 出了换 GPT-5。
> **车不用换**。
> 市面上的 AI 工具大部分绑死自己家模型，我们绑你。"

### 第 1:20-1:50 · 演示 3 · 换 Pack（亮点 3 · 行业可换皮）

> 我："如果我做的是财务报表分析呢？切到财务方向盘"

AI 应回：

```
✓ 已切换 Pack：跨境电商 → 财务（demo）
✓ 重启 Hermes Gateway 中…
✓ 完成。Home 已更新。

  • 隐藏：广告 / 库存 / 差评卡片
  • 加载：现金流 / 应收应付 / 损益表卡片

👀 [回到 Home 看新卡片 →]
```

**对客户说**：

> "同一个软件、同一个二进制。我给你的是跨境电商方向盘，
> 给我隔壁财务朋友的就是财务方向盘。
> 装一次、永不打架。"

> ⚠️ 财务 Pack 现在是空的，演示前要么造个最小版，要么把这一段换成"我给跨境电商朋友演示就只装这一个 Pack"。

### 第 1:50-2:20 · 演示 4 · 离线（亮点 2 · 数据 0 外发）

> 我："我现在拔了网线给你看"

物理拔网线 / 关 wifi。

AI 应回（继续工作）：

```
✓ 检测到离线
✓ 自动切换到本地模型：MLX-Qwen-7B
✓ 当前状态：本地运行 · 离网 · 数据 0 外发

[ 🟢 离线模式 ]   ← 顶部出现绿色徽章
```

**对客户说**：

> "你最敏感的就是数据。
> 我们这个软件设计上**就是离线优先的**。
> 哪怕没网，本地小模型继续工作。
> 你的报表、客户名单、利润数据，**没有一个字节发出过你的电脑**。"

### 第 2:20-2:50 · 演示 5 · 你试试（让客户开口）

把鼠标 / 麦克风递过去：

> "你来。说一句话试试。
> 比如说**'帮我看看断货的 ASIN'**，或者**'给我看本月利润'**。"

**重点**：闭嘴。让客户**自己说话给 AI**。

不管 AI 答得好不好，**客户开口 = 演示成功**。

### 第 2:50-3:00 · 收尾

> "今天看到的所有数据是 demo 假数据。
> 你愿意的话，发一份你自己的真实报表给我，
> 一周后我把你的真实数据接进来再演示给你看。
> **0 块钱、0 风险、不签任何合同**。"

闭嘴。让对方说话。

## 演示成功的 KPI

| 指标 | 标准 |
|---|---|
| 客户至少一次说"哦"或凑近屏幕 | ≥ 1 次 |
| 客户问"价格多少" | 加分 |
| 客户主动说自己的真实数据格式 | 大加分 |
| 客户主动问"什么时候能装" | 强信号 |
| 客户提建议改某功能 | 强信号 |
| 客户问"还有什么 Pack" | 弱信号（说明前面打动了） |

3 项达到 = 种子用户。
